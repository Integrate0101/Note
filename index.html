사용 방법


1. 위 그림과 같이 drawing area를 클릭한다. (만약 drawing area를 삭제하고 싶다면 해당 영역에서 우클릭 후 delete 하면 된다.)


2. 추가된 격자무늬 영역을 클릭 후 화면 왼쪽에 있는 메뉴를 이용하여 그림을 그린다.


3. 대충 그림 그린 후 그린 영역을 드래그하여 좌측 상단 menu -> export selection latex 를 누르면 코드가 나온다. 이 코드를 복사하여 latex에서 그림을 붙이고 싶은 위치에 붙여넣기 하면 된다.
